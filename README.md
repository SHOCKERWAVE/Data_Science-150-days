                                                                      ### Data_Science-150-days ###


    # Day 1 - Python Basics
    Learned:
    -Variables (boxes to store info)
    -Data types (labels on the boxes)
    -Printing (showing what's inside)
    -Type conversion (changing labels)


    # Day 2 - Python Lists & Tuples  
    Learned:
    -Lists = editable collections (you can add, remove, change items)
    -Tupes = fixed collections (you cannot modify them)
    -Lists are mutable , tuples are immutable   


     # Day 3 - Python Dictionaries & Sets
    Learned:
    - Dictionary = key -> value pairs (like word -> meaning)
    - Keys must be unique
    - Values can be anything
    - Sets = unique items only  (no duplicates)
    - Practiced adding, updating, and removing items


     # Day 4 - Python If / Else
    Learned:
    - If = check a condition
    - Else = run when condition is false
    - Elif = check more conditions


    # Day 5 - Python Loops (For & While)
    Learned:
    -For loop = repeat a fixed number of times
    -While loop = repeat until condition becomes false
    -break = stop the loop early
    -continue = skip one step


    # Day 6 - Python Functions
    Learned:
    - Functions = mini-machines in python
    - def keyword to create functions
    - return keyword to send back results
    - Default value(arguments)


    # Day 7 - Python Modules & Importing
    Learned:
    - Modules = toolboxes in python
    - import keyword to use modules
    - math, random, datetime modules
    - How to import specific functions


    # Day 8 - Python File Handling
    Learned:
    - How to open, read, write and append files
    - File modes: 'r', 'w', 'a'
    - using 'with' for safe file handling


    # Day 9 - Python Error Handling
    Learned:
    - try = code that might fail
    - except = handle the error
    - else = runs if no error
    - finally = always runs


    # Day 10 - Python OOP (Classes & Objects)
    Learned:
    - Class = blueprint
    - Object = created from class
    - Attributes = data inside object
    - Methods = actions inside object
